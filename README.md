# SberAI Personal Communication Assistant

Проект по персонализации языковых моделей SberAI.

```
# 1. Клонируем ваш стиль из Telegram
python main.py --mode clone

# 2. Обучаем персонализированную модель
python main.py --mode train

# 3. Тестируем демо
python main.py --mode demo

# 4. Запускаем Telegram бота
python main.py --mode bot

# 5. Или запускаем веб-интерфейс
python main.py --mode web

```
